# MeowCore

Welcome to **MeowCore**, a high-performance, easy-to-use desktop application. MeowCore is a robust and efficient checker utility for Microsoft and Minecraft accounts, complete with extensive statistical tracking, proxy support, and a seamless login system.

## ✨ What does it do?

MeowCore allows you to quickly and efficiently validate large lists of Microsoft and Minecraft accounts. It automatically checks each account's status, captures profile information, and fetches detailed in-game statistics, saving you hours of manual work. 

## 🚀 Key Features

- **Advanced Checking Engine**: Check multiple accounts at the same time for incredibly fast validation using multi-threading.
- **Detailed Statistics & Captures**: Automatically tracks and categorizes accounts based on:
  - Hits (Working Accounts)
  - Account Access Types (NFA, MFA)
  - 2FA (Two-Factor Authentication) Status
  - In-game stats: Hypixel Network Level, Skyblock Networth, SkyWars, UHC, Pit, Cloaks, and more!
- **Proxy & Combo Support**: Effortlessly load your account combos (`email:password` format) and proxy lists. Built-in support for HTTP and SOCKS proxies to keep your checks running smoothly and avoid rate limits.
- **Secure Login System**: Dual login system offering a free tier via normal accounts (Email/Password) and a premium tier unlocked via your unique License Key!
- **Sleek Interface**: A beautifully crafted, modern application window designed to be straightforward and intuitive.
- **Auto-Updates**: Never worry about downloading the latest version manually. MeowCore automatically detects, downloads, and applies the latest updates on launch.

## � How to Use

1. **Launch the App**: Simply double-click and run the `MeowCore.exe` file.
2. **Sign In**: Log in using your MeowCore account credentials or enter your premium License Key.
3. **Load Data**:
   - Click to load your **Combos** text file (containing the accounts you want to check).
   - Click to load your **Proxies** text file (to prevent rate-limiting from the servers).
4. **Configure Settings**: Adjust the number of threads (speed) and toggle proxies on or off within the app interface.
5. **Start Checking**: Hit the **Start** button and watch the magic happen! The application will automatically sort and save valid accounts.

## ⚙️ Results & Output

All valid accounts and their detailed statistics are neatly organized and saved into dedicated, timestamped result folders in the same directory as the executable. This makes it incredibly easy to review your "hits" and account details at any time.
